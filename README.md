# discord-nitro-token-sorter
Tool that sorts discord tokens depending on whether the tokens have nitro or not

# How to use
1. Run ```setup.bat```
2. Paste your tokens into ```tokens.txt``` (one per line)
3. Run ```discord-token-sorter.py```
